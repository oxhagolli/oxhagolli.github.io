Alpha Pill Pal

My most recent hackathon project was at UB Hacking (University at Buffalo), where my team and I created a web service that helps patients and their doctors set text or call reminders to take their medicine when needed. This service allows the prescriber to set the reminder without the user having to do anything. We use the twilio API to send texts and calls. This project won the Best Health Hack prize. Languages & frameworks: Python, Django, Twilio, HTML, CSS

MYOwnLife

At HackPSU (Pennsylvania State University), my team and I created MYOwnLife which is a project that uses a MYO armband and a seizure detection algorithm that we wrote to help people that are suffering from epilepsy. We take in the input from the MYO armband, process it and if we detect anything outside of the normal, we activate a "risk event" on supplementary app that we wrote. If the event is activated, the app will ask the user if everything is okay, and if the user does not respond within 5 seconds, we call a predetermined number which could be a best friend or a family member. The user also has the option to immediately call 911. We won the IEEE prize for this project. Languages & frameworks: Java, C, Python, MySQL, PHP, Android, MYO

UFC Predictor

This is a personal project that started off orginially proposed as a business idea by Olsjon Baxhija. I wrote a python script that retreives match predictions from the betting companies and runs a fairly simple analysis on past matches, predicting the winner and subsequently whether you should bet or not. The predictions passed testing on an extended data set and yielded very favorable returns. Olsjon and I have agreed to open source this project soon.
