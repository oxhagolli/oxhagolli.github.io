Chalk

Have you ever had so many things to do that at some point that you think "What do I do first"? That is exactly the problem that a few friends and I tried to solve at HackBU (Binghamton University). Basically, you can create your own prioritized schedule and tell our program when you're free and when do you need to do everything by and we sort out your schedule so that you can get everything done in time and without worrying about what to do first. This project won third prize at the hackathon. I worked mainly on the backend. Languages & frameworks: Python, Django, HTML, CSS

Alpha Pill Pal

My team and I created a web service that helps patients and their doctors set text reminders to take their medicine when needed. This service allows the prescriber to set the reminder without the user having to do anything. We use the twilio API to send texts. This project won the Best Health Hack prize at UBHacking (University at Buffalo). I worked primarly on the backend. Languages & frameworks: Python, Django, Twilio, HTML, CSS

MYOwnLife

At HackPSU (Pennsylvania State University), my team and I created MYOwnLife which is a project that uses a MYO armband and a seizure detection algorithm to help people that are suffering from epilepsy. We take in the input from the MYO armband, process it and if we detect anything short of normal, we activate a "risk event" on a supplementary app that we wrote. If the event is activated, the app will ask the user if everything is okay, and if the user does not respond within 5 seconds, we call a predetermined number. The user also has the option to immediately call 911. We won the IEEE prize for this project. I worked on the app and the detection algorithm. Languages & frameworks: Java, C, Python, MySQL, PHP, Android, MYO

UFC Predictor
